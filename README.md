# LifePath

Eine visuelle App zur Dokumentation des eigenen Lebens über mehrere parallele Zeitstrahlen (z. B. Karriere, Familie, Reisen). Erinnerungen werden als Events mit Text & Medien gespeichert und in einer zoombaren „Sternen/Universum“-Ansicht exploriert. Eine KI fasst Lebensphasen zusammen und hebt Highlights hervor. Für Menschen, die ihr Leben reflektieren & dokumentieren möchten.

- Kosmos Optik: Zeitstrahl aus Sterne
- Rein-/Rauszoomen wie bei Google Maps
- Fotos & Videos importieren
- Zeitstrahlen teilen

Tech Stack:

- Frontend: Vue 3 + Vite + JavaScript
- Visualisierung: WebGL mit three.js
- State: Pinia
- Backend: API + Medien-Storage (Datenbank?)
- KI: Zusammenfassungen, Highlights
